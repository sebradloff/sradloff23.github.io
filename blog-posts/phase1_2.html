An Easy Guide to Setting up Your Terminal

Cohort: Bobolinks, Chicago, July 2014

I've seen several great terminal setups but this one by Remy Sharp was my favorite. I was all excited to set up my new terminal, but I kept running into issues throughout the process. Being relatively new to the command line, it would have been great to have a step-by-step tutorial for customizing my terminal. I decided to write that tutorial here.

The first step is to download iTerm 2 iTerm is highly customizeable, offers the cmd+f option like in Chrome, and much more.

Once you've downloaded iTerm and installed any updates, you should download the Inconsolata Powerline font, by clicking the raw button on the right hand side. Once the file has downloaded, double click to open it, which should automatically open it in Font Book. Click 'Install Font' in the bottom right corner of the window. Open 'iTerm Preferences' => 'Profiles' => 'Text', and change the 'Regular Font' to 'Inconsolata-dz for Powerline' and size 15. Keep iTerm open.

Now we'll download the Solarized Dark theme here. Ctrl + click and save the file to your Desktop. In iTerm Preferences, switch to the 'Colors' tab and click the 'Load Presets' dropdown. Select 'Import', and find the downloaded Solarized Dark theme file. Click on the 'Load Presets' button once again and select the Solarized Dark theme. If you open up a new iTerm window, the background should be dark blue.

Continuing in the 'Colors' tab, uncheck the 'Smart cursor color' box and click on the colored box next to 'Cursor'. Set the color using 'Color Sliders' => 'RGB Sliders' to 209, 58, and 138.

Next you'll need to download 'oh-my-zsh', which is responsible for managing multiple themes and skins for your terminal. You can download it from the command line by copying and pasting 'curl -L http://install.ohmyz.sh | sh' OR 'wget --no-check-certificate http://install.ohmyz.sh -O - | sh' . Either set of commands should work.

Go to the users section on your Mac (open a Finder window and click on the house icon with presumably your name next to it). You may or may not have hidden files set as invisible. If you see a “dim” version of a '.oh-my-zsh' older, your hidden files are visible. If not, you should copy and paste the following commands in your terminal to make those files visible: 'defaults write com.apple.finder AppleShowAllFiles TRUE',press enter, then 'killall Finder'.You can reverse this process after this tutorial by typing 'defaults write com.apple.finder AppleShowAllFiles FALSE', press enter, then 'killall Finder'.

Right click on Remy's customized agnoster skin, and save the file to your Desktop. Right click the downloaded file and select 'Get Info'. Go to the 'Name & Extension' section and delete the '.txt' extension from the file name. A window should pop up asking if you're sure you would like to use the '.zsh-theme', which you should click. Make a note of the file name.

Now locate the '.oh-my-zsh' folder and navigate your way to the 'themes' folder. Drag Remy's customized agnoster skin file into the 'themes' folder.

Going back out to your user tab in Finder (the one with the house), locate the '.zshrc' file. Right click the file and open it with Sublime. Look for 'ZSH_THEME', should be on line 8, and set the variable that's in between string to "remy". This sets the theme of to the skin you downloaded and placed into the themes folder. If you want to switch to any other theme, simply add the name of the skin, minus the extension, into between strings in the 'ZSH_THEME' variable. Once set, save the file and close it.

Now go back to the user tab in Finder and search for your 'bin' folder. If you do not have a 'bin' folder in your User tab, you can simply add a new folder and name it 'bin'. You'll want to download this file by right clicking and saving the file in your 'bin' folder. This file will show you how much battery you have left in your terminal, and is necessary to run the theme properly. Open up a terminal window to see if you have the proper permissions. Navigate to your 'bin', 'cd bin', and then run 'ls -l'. You should now enter 'chmod +rwx batcharge.py' to change the file mode and its permission to run. You also need to open up the 'batcharge.py' file in Sublime. Copy the line '# coding=UTF-8' and paste it after the '#!/usr/bin/env python' on the first line, meaning you should have the same code twice.

Open up a new iTerm window and that should be it! I hope you found this tutorial informative and easy to follow.
