JavaScript and Web Development

Why is JavaScript popular in web development?

JavaScript is on the rise as on the most popular programming languages in the world. What has led to this spike on popularity? I do not think there is any one reason, but there are several factors that certainly helped.

For starters, JavaScript is a great first programming language. It was developed as a client side scripting language, meaning all the work is done in the browser rather than an external server, and has been dubbed "fairly easy" to learn. While it might be a simple language, one can write plenty of advanced scripts while accounting for browser differences, things can become quite complex.

HTML5 can now support more complex JavaScript methods and functions. Real-time 3D graphics, geolocation and multitouch capabilities are only several JavaScript methods which HTML5 provides the structure.

Finally, JavaScript has the ability to meet rapid development demands while retaining a great user experience. This has it being pushed as the language used in mobile development platforms, coming from both Apple and Google.

Pros of JavaScript:

1. FAST!
Being used in client-side operations is very fast, because it does not need to contact a server for an answer, it runs immediately. It is ready as soon as the browser is!
2. Easy to add on
While building other parts of your code, you can run the rest of the code just fine. This is because of its simple load and go delivery. It only throws errors while executing a particular path of the program.
3. Everything is an object
You can add properties to any object at any time, which allows you to access properties through the dot operator.

Cons of JavaScript

1. Web browser compatibility
Different web browsers will interpret JavaScript slightly differently, which can make it a pain to write the code for several different browsers.
